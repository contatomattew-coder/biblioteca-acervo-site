# Como subir arquivos no site

O site é estático, então ele não envia arquivos diretamente pelo formulário do site. O jeito grátis e simples é usar o GitHub Release como armazenamento.

## Passo a passo

1. Entre no GitHub.
2. Abra: https://github.com/contatomattew-coder/biblioteca-acervo-site/releases/edit/v1-media
3. Na área de Assets, arraste os novos arquivos.
4. Salve o release.
5. Volte no site e clique em “Atualizar acervo”.

## Organização de nomes

Use nomes claros para ficar bonito no site:

- `video_30.mp4`
- `evento_praia_01.jpg`
- `biblioteca_livro_01.png`
- `acervo_noite_02.mov`

O site não mostra a data de adição nos cards.
